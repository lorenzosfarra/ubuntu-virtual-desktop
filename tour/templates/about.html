About ...


WARNING: THIS PROJECT IS IN DEVELOPMENT. IT'S ABSOLUTELY NOT COMPLETED AND THERE ARE BUGS / ERRORS. FOR NOW, IT DOESN'T SUPPORT IE6!


This project tries to show an Ubuntu interactive presentation through the web.

Starting the tour in the main page, you can see a simple Ubuntu desktop, with some icons and the menu.
You can browse the menu and launch the applications. Each application could be represented by:
  1. A screenshot of that application: a launched application's screenshot, showing the application in action;
  2. A more-interactive widget: in some cases (like Mozilla Firefox), you can "try" the application. Obviously, this is only a simulation.
Anyway, this applications are "started" in windows-like widget, and you can move, minimize, close and maximize/restore them just like in a real desktop.
